# Shared

A ready to use while loop that never break instead of generating multiple loops every time which could lead to the overheat

# lib.Repeat(func)
```lua
lib.Repeat(func)
```

## Parameters
- func `function`: The function to repeat

## Returns
- `void`

## Examples

```lua
lib.Repeat(function()
  print("This will never break")
end)
```
