# Installation
## Typescript
### Depend
```json
"@trippler/tr_lib": "latest",
```
### Install
```bash
npm i @trippler/tr_lib
```
## Lua
### 1. Download
<Cards.Card
  icon={<DownloadIcon />}
  title="GitHub Release"
  href="https://github.com/tripplerscripts/tr_lib/releases/latest/download/tr_lib.zip"
/>
### 2. Import
* **lua**
```lua filename='fxmanifest.lua'
shared_scripts {
  '@tr_lib/init.lua',
}
```
