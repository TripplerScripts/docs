# Shared
Load a file's content so you can store it and use it's content without executing it

## lib.load

```lua
lib.load(path)
```

### Parameter
* path `string`: the path to the file you want to load, the file's extention is optional

### Return
* `...any`: the content of the file, usually an array of tables

### Example

```lua filename='fxmanifest.lua'
files {
    'client/data.json',
    'shared/config.lua'
}
```

```lua
local data = lib.load('client/data')
local config = lib.load('shared/config.lua')
```

### Notes
* The file you want to load must be in the (`files` | `file`) (table | string) of your `fxmanifest.lua`
