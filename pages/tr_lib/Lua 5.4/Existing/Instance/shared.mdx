# Shared
Instance existing is a module to await for a certain entity to exist physically

## lib.awaitInstanceExisting

```lua
lib.awaitInstanceExisting(entityHandle, entityNetId, timeout)
```

### Parameters

* entityHandle?: `number` The entity to await.
* entityNetId?: `number` The network id of the entity.
* timeout?: `number` The timeout in milliseconds to wait for the entity to exist.

### Returns
* entityHandle `number | false`: The handle of the entity.
* entityNetId `number` | `nil`: The network id of the entity.

## Callouts
<Callout type='info'>
* Please be aware that at least you need to pass an entity or a net id and don't pass them both aswell
</Callout>
